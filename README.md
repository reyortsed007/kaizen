This project is a simple and user-friendly Task Manager web application built using HTML, CSS, and JavaScript. It allows users to efficiently manage their daily tasks and stay organized with time-based reminders.

Key functionalities include:

Adding tasks with a clear and intuitive interface

Marking tasks as completed using a toggle option

Removing tasks when they are no longer needed

Setting a specific reminder time for each task

Triggering an alert message and sound notification when the reminder time is reached

The application demonstrates core front-end development concepts such as DOM manipulation, event handling, and basic time-based logic using JavaScript, making it a solid foundation project for learning and expanding into full-stack development.
